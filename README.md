# web_aesthetic
GlowUp Path adalah platform pendamping transformasi diri yang menyeimbangkan perawatan diri, kebugaran, dan pola hidup positif. Dengan tagline "Your Journey to Radiance", platform ini mengajak individu menemukan cahaya dari dalam—bukan hanya penampilan luar, tetapi juga kesehatan mental dan kebiasaan sehari-hari.
